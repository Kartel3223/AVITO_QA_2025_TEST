# TESTCASES

Базовый URL: `https://qa-internship.avito.com`

Ручки:
- `POST /api/1/item` - создать объявление
- `GET /api/1/item/:id` - получить объявление по id
- `GET /api/1/:sellerID/item` - получить все объявления пользователя
- `GET /api/1/statistic/:id` - получить статистику по объявлению

---

## TC-01 Успешное создание объявления

**Шаги:**
1. Отправить `POST /api/1/item` с телом:
   - `sellerID` - число в диапазоне 111111-999999,
   - `name` - непустая строка,
   - `price` - положительное число,
   - `statistics` - объект с полями `likes`, `viewCount`, `contacts` = 0.

**Ожидаемый результат:**
- Статус `200 OK`.
- В теле есть поля `id`, `sellerId`, `name`, `price`, `statistics`.
- `sellerId` равен отправленному `sellerID`.
- `name` и `price` совпадают с отправленными.

---

## TC-02 Получение объявления по id

**Шаги:**
1. Создать объявление (TC-01), взять его `id`.
2. Отправить `GET /api/1/item/{id}`.

**Ожидаемый результат:**
- Статус `200 OK`.
- В ответе объявление с таким же `id`.
- Поля `sellerId`, `name`, `price` совпадают с созданным объявлением.

---

## TC-03 Получение всех объявлений по sellerID

**Шаги:**
1. Создать несколько объявлений с одним и тем же `sellerID`.
2. Отправить `GET /api/1/{sellerID}/item`.

**Ожидаемый результат:**
- Статус `200 OK`.
-- Ответ - массив.
- В массиве есть все созданные объявления.
- У всех элементов поле `sellerId` равно запрошенному `sellerID`.

---

## TC-04 Получение статистики по объявлению

**Шаги:**
1. Создать объявление, взять его `id`.
2. Отправить `GET /api/1/statistic/{id}`.

**Ожидаемый результат:**
- Статус `200 OK`.
-- Ответ - массив.
- В первом элементе есть поля `likes`, `viewCount`, `contacts`.
